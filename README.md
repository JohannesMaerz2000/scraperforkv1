# Tennis.de Match Scraper (scraperforkv1)

A Chrome extension for scraping match data from tennis.de and syncing it with a Supabase database.

## Features
- Scrapes match results from player profiles.
- Supports tournament and league search.
- Syncs directly with Supabase.

## Setup
- Replace the Supabase credentials in `config.js` with your own if you haven't already.

## Development
- Load the folder as an unpacked extension in Chrome.
